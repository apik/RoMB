Romb

Representation of Mellin-Barnes

Download as .zip Download as .tar.gz View on GitHub

Use prepackaged source

Get the packaged source code release from Download Packages section at https://github.com/apik/RoMB/downloads

$ tar -jxvf RoMB-x.y.z.tar.bz2
$ cmake CmakeLists.txt
$ make

Use code repository

Get complete code repository by git clone

$ git clone git://github.com/apik/RoMB.git
$ cmake CmakeLists.txt
$ make
